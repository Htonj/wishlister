## Wishlist Project README

# Wishlist

This project is an application for creating and managing wishlists. Users can add, edit, and delete items in their wish list.

### Description

Wishlist is an application that allows users to manage their desires and preferences. It enables users to store a list of desired products or services and share this list with friends and family.

### Installation

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Htonj/wishlister
   ```
2. Navigate to the project directory:
   ```bash
   cd wishlist
   ```
3. Open the project in your preferred C# IDE (like Visual Studio).

4. Restore the dependencies if necessary. You can do this using:
   ```bash
   dotnet restore
   ```

5. Run the application:
   ```bash
   dotnet run
   ```


### Technologies Used

- **C#**: Programming language used for the application.
- **.NET Core**: Framework for building the application.
- **Entity Framework**: To manage database interactions.
- **SQLite**: Lightweight database for storing wishlist items.

### Usage

1. Start the application.
2. Use the provided UI to add items to your wishlist.
3. Edit or delete items as needed.
4. Optionally, share your wishlist link with others.
